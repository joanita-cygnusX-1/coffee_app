# User Stories – Coffee App

This file contains all user stories for the Coffee App.  
As the project grows or becomes dynamic, the user stories will be managed here.

---

## Story 1: View App Name
**As a** customer  
**I want to** see the name of the restaurant/app  
**So that** I know where I am ordering from  

**Acceptance Criteria:**
- Landing page displays app name
- Prices are visible
- A user ID is generated for order retrieval

---

## Story 2: View Menu
**As a** user  
**I want to** see the menu  
**So that** I can choose my drink  

**Acceptance Criteria:**
- Menu loads
- Menu displays items + prices
- User can select items

---

## Story 3: Add Item to Cart
**As a** customer  
**I want to** select items and quantity  
**So that** I can build my order  

**Acceptance Criteria:**
- User can input item and quantity
- Item is added to cart
- Cart totals update accordingly

---

## Story 4: View/Edit/Delete Cart
**As a** customer  
**I want to** review and modify my order  
**So that** I can make changes easily  

**Acceptance Criteria:**
- Cart can be viewed
- Items can be edited or removed
- Cart totals update dynamically

--- 

## Story 5: Checkout
**As a** user  
**I want to** checkout and choose payment  
**So that** I can complete my order  

**Acceptance Criteria:**
- Debit/credit card options
- Scan QR or OTP option
- Option to print receipt or “Go Green”
